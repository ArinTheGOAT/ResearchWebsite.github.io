About

We are Arin, Noah, Hershil, and Andrei—friends who wanted one calm place to host writing we stand behind: preprints, course projects, lab notes, and anything in between.

This version of the site is plain HTML, CSS, and a little JavaScript. There is no build step. You can open index.html locally or drop the folder on any static host (including GitHub Pages).

How to publish something new

  1. Copy an existing file in articles/ and rename it.
  2. Edit the title, date, author, and body in that HTML file.
  3. Add a matching entry to js/papers-data.js so it appears on Home and Papers.
