Welcome to our manuscript shelf

Why this site exists and how we use it.

This is a lightweight home for things we are willing to share with the world: course papers, side projects, half-formed ideas that still deserve a URL.

What to expect

  • Honest drafts. Not everything here is peer-reviewed; we label context in the text when it matters.
  • Slow updates. We publish when we have something worth reading, not on a schedule.
  • Plain files. Each piece is an HTML file in the repo you can track like any other document.

Thanks for stopping by.
